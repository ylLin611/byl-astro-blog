# Astro Blog Test

This is a test of the Astro Blog.

## Getting Started

To get started, clone this repository and install the dependencies:

```bash
git clone https://github.com/astro-community/astro-blog-test.git
cd astro-blog-test
npm install
```

Then, run the development server:

```bash
npm run dev
```
